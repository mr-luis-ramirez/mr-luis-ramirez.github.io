Control Flow and Loops

For the Control Flow and Loops, I am going to use baking to describe these with a simple metaphor so that you can have a better idea of these two computer jargons.
Control flow is like as I mentioned from the above statement is like baking, there are steps that you have to do for it to come up with a taste and form of a cake, it can be sequential so that the ingredients with react with one another and some procedure can be done separately so that after the main cake is bake, we can add the icing or other glazing for the cake.

Loop in programming is the repeated instruction, like whipping the flour with egg and butter until thick, so you keep on mixing until you get the desired result. In programming you can have a value that you are aiming so that the procedure would stop once you get that value.

In programming usually may have a target value or a scenario like true, false etc., in order for the process to stop. In baking you set a value to aim for a certain quality to terminate the process, like maybe the thickness or consistency of the mixture.

DOM (Document Object Model)

A programming interface for HTML, allowing your code to access and manipulate the document ‘s structure, style and content. This process is like a dog, if you can create your own dog having the basic facilities of a normal dog, it would have methods sort of like an action and have attribute like characteristic. You access these and you can change them to your own. If you like to have it a brown coloured skin, then you can say change skin colour to brown. For DOM you can access a HTML tag and can change its properties.

Array and Objects

Both Array and Objects are container similar to a box but the difference would be for object you can define the properties, like if you define a person object. The person object may have name, age, height and weight and other information. This will be the basis when you fill up the information for this storage, you need to give the value to the properties you define when you created this storage.
